# Amazon Clone Website 🛒

## 📌 About The Project

This project is a clone of the Amazon homepage built using **HTML** and **CSS**.  
The purpose of this project was to improve my frontend web development skills by recreating a real-world e-commerce website interface.

I designed different sections like:

- Navigation Bar
- Search Bar
- Hero Section
- Product Cards
- Shopping Sections
- Footer Section

This project helped me understand how professional websites are structured and styled.
